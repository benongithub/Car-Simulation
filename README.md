## Installation

- Clone this Repository
- Create new Virtual Environment: `python3 -m venv crsm`
- Install Requirements: `pip3 install -r rquirements.txt`
- Run Main Program: `python3 main.py` or
- Run Notebooks `jupyter notebook`
